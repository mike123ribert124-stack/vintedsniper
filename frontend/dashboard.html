🔑 Connexion Vinted

Pour que le bot puisse acheter, il a besoin de ton cookie de session Vinted. Ce cookie est stocké de manière chiffrée.
1. Connecte-toi sur vinted.fr dans ton navigateur
2. Ouvre les outils dev (F12) → onglet Application → Cookies
3. Copie la valeur de _vinted_fr_session
